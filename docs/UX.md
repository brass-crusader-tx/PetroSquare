# PetroSquare UX/UI Design Language Constitution v1.0

## Core Principles
1.  **Decision-Centric Design**: Every interface element must support decision-making.
2.  **Engineering-Grade Legibility**: Use monospace fonts for data, clear contrast, and precise labels.
3.  **Transparency**: Show units, data sources, and confidence intervals.
4.  **Vendor Neutrality**: Design should not bias towards specific vendor tools.

## Visual Language
- **Theme**: Dark Mode First.
- **Backgrounds**: Slate 900 (`#0F172A`) for app background, Slate 800 (`#1E293B`) for surfaces.
- **Typography**: `JetBrains Mono` for data/code, `Inter` for UI text.
- **Colors**:
    - Primary: Blue 500 (`#3B82F6`)
    - Text: Slate 50 (`#F8FAFC`)
    - Muted: Slate 400 (`#94A3B8`)
    - Border: Slate 700 (`#334155`)

## Component Guidelines
- **Buttons**: Clear hierarchy (Primary, Secondary, Danger).
- **Cards**: Used for grouping related metrics or controls.
- **Charts**: Must include legends and axis labels.
